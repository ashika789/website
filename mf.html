    Mobile Framework

    An abstraction layer that is built upon the existing iOS framework to provide common, low-level software components and services that can be leveraged by all iPad applications developed within GWIM.


    Capabilities

    Instrumentation

    Mobile Framework collaborates with Sym Instrumentation to maintain logs for:
    • Error Message, Warning Message, Information Message, Trace Information [Helpful for remote debugging]
    • User Analytics [Usage Analytics]

    Proxy Services

    • Generic component to consume services (both JSON and SOAP).
    • Built in Service Usage Instrumentation and Data Capturing
    • Supports both Sync and Async calls.

    Authentication

    Integrated with Security Framework to authenticate user credentials to access the application and maintain security access.

    MF Services (Version Integrity Check & Service Resolver)

    Feature to check the application integrity and version update

    Local Context & Event Notification

    Context manager provides an in memory data store for sharing state variables between applications

    Adaptive Design Framework for iOS

    iOS library that enables web application hosted inside Desktop Container to bring in the flexibility of iOS platforms/native specific features, look & feel and transitions

    Benefits


    iOS Native Experience

    The web application can bring in the iOS native experience to the user with native power and capability.

    Configurable

    Native widgets/controls are configurable using JSON files by setting appropriate properties.

    Supports View Transitions

    Supports various iOS native view transitions such as flip, fly-in, drop-down, etc.

    New iOS native widgets

    Provides the ability for the development team to create new iOS native widgets/controls that can be leveraged by web application.

    Scalable

    Can be scaled to other Mobile Platform such as Android and Win 8
