UI Framework

UI Framework provides a framework and a common set of services to facilitate building Device Agnostic, Mobile First Modern Single Page Application development. All the necessary common service calls are abstracted from the application team so that they can interact with one set of API, be it they are running on the mobile device or a desktop or a browser

Mobile Framework

An abstraction layer that is built upon the existing iOS framework to provide common, low-level software components and services that can be leveraged by all iPad applications developed within GWIM

Adaptive Design Framework

iOS library that enables web application hosted inside Mobile Container to bring in the flexibility of iOS platform/native specific features, look & feel and transitions
